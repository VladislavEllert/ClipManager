# ClipManager

Менеджер истории буфера обмена для macOS — аналог `Win+V` из Windows, которого в macOS нет из коробки.

Фоновое приложение в строке меню: по глобальному хоткею открывает компактное окно
в стиле Liquid Glass со всей историей копирований (текст, фото, файлы), даёт выбрать
любой элемент и вставить его в активное приложение.

## Зачем

В macOS буфер обмена хранит только последнее скопированное. Чтобы перенести
несколько фрагментов, приходится повторять «копировать → вставить» по очереди.
ClipManager хранит историю и позволяет листать её и вставлять что угодно.

## Возможности

- Захват из буфера: **текст, RTF, изображения, файлы**
- История **на диске** — переживает перезапуск и перезагрузку
- Глобальный хоткей (по умолчанию `⌘⇧V`), настраиваемый
- **Авто-вставка** выбранного элемента в активное окно
- Настраиваемый лимит истории **1–100**
- **Поиск** по истории
- **Избранное** (закрепление элементов — не вытесняются лимитом)
- **Вставка без формата** (`⌥↵`)
- **Пропуск паролей** (игнорирует буфер от менеджеров паролей)
- Переключатели фич **прямо в окне** + окно **изменяемого размера** (запоминает размер)
- **Автозагрузка** при входе в систему
- Живое обновление списка при открытом окне

## Требования

- macOS 26 (Tahoe) или новее
- Xcode 26+

## Запуск

```bash
git clone git@github.com:VladislavEllert/ClipManager.git
cd ClipManager
open ClipManager.xcodeproj
```

В Xcode:

1. Выбери проект **ClipManager** → target **ClipManager** → **Signing & Capabilities**
   → включи **Automatically manage signing** и выбери свою команду (Apple ID).
   Стабильная подпись нужна, чтобы разрешение Accessibility не слетало между сборками.
2. Нажми **Run** (`⌘R`).
3. Появится иконка в строке меню (в Dock приложения нет — это фоновый агент).
4. Для авто-вставки выдай разрешение: **System Settings → Privacy & Security →
   Accessibility → включи ClipManager**, затем перезапусти приложение.

## Использование

| Действие | Клавиши |
|----------|---------|
| Открыть/закрыть окно истории | `⌘⇧V` |
| Навигация по списку | `↑` / `↓` |
| Вставить выбранное | `Enter` |
| Вставить без формата | `⌥Enter` |
| Закрыть окно | `Esc` |
| Закрепить / удалить элемент | правый клик по элементу |

Переключатели фич (поиск, избранное, без формата, пропуск паролей) и шестерёнка
настроек — в правом верхнем углу окна. Лимит истории, хоткей и автозагрузка — в
настройках (`⌘⇧V` → шестерёнка, или меню иконки в строке меню).

## Разрешения

- **Accessibility** — нужно только для авто-вставки (эмуляция `⌘V` через `CGEvent`).
  Чтение буфера и хоткей работают без него.

## Стек

Нативный **Swift + SwiftUI/AppKit**, без внешних зависимостей. Глобальный хоткей —
на Carbon (`RegisterEventHotKey`). Хранение — JSON-метаданные + PNG-блобы в
`~/Library/Application Support/ClipManager/`.

## Ограничения

- Файлы хранятся по пути (не копируются) — перемещённый/удалённый оригинал
  не вставится. Текст и фото не затрагиваются.
